# Scripts Index

This folder has many one-off utilities. The scripts below are the core maintenance path for the website.

## Core maintenance scripts
- `scripts/download_torvik_2026.sh` — Download Bart Torvik 2026 CSV
- `scripts/build_data.R` — Build `public/data/season.csv` + `public/data/career.csv`
- `scripts/realgm_scrape.py` — Scrape RealGM international leagues
- `scripts/import_2026_international.py` — Update `2026_records.csv`
- `scripts/import_realgm_leagues.py` — Update `internationalplayerarchive.csv`

## Other scripts
Many other scripts are experimental or one-off. If you need a cleanup/move pass later, we can archive those into `scripts/_legacy`.
